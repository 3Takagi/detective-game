# AI驱动侦探剧场 Web 小游戏

## 入口

直接打开 `index.html` 即可运行。

## 当前玩法

- 当前剧情已根据 `案件文本_详细版.docx` 重写，主线包含“星核系统”代码失窃、商业间谍身份、茶水间纸杯、匿名转账和三人交叉证词。
- 游戏以开场封面进入，主界面只显示当前章节场景、叙事文本和行动选项。
- 玩家按时间线推进剧情：20:30、21:10、21:15、21:25、21:35、22:10。
- 案卷、证据、人物、结案都以弹窗形式打开，避免把所有内容摊在同一页。
- 推进到后续章节后，玩家可以点击顶部时间节点，或使用“上一幕/下一幕”，回到任何已经解锁的完整章节页面。
- 所有物证都以中性标签呈现，不直接标明是否为干扰信息，玩家需要自行判断证据重量。
- 结案不需要输入长文本，改为选择式推理：真凶、协助者、死因、密室形成方式、关键证词矛盾、证据链闭合。
- “环境声”改为更克制的雨夜/空调底噪和轻提示音，默认关闭。

## 正确结论

林薇亲手下毒杀害陈峰，苏晴协助发送短信拖延时间线。作案方式为氰化物咖啡；密室由备用钥匙、反锁办公室和安全通道离开共同制造。
